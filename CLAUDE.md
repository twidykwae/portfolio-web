# Design Context

Brand-register personal portfolio for Twidy Kelvin Kwae. Primary audience: grad school admissions and fellowship committees. Conversion surface: project links to GitHub.

See [PRODUCT.md](./PRODUCT.md) for users, brand personality, anti-references, and the five design principles that govern all design work in this repo.

See [DESIGN.md](./DESIGN.md) (when present) for the visual system: colors, typography, components, layout tokens.

When designing or modifying UI in this project, run `/impeccable` commands so the work stays aligned with PRODUCT.md and DESIGN.md.

## Project layout

- `client/react/` — Vite + React 19 + Tailwind 4 frontend.
- `backend/` — API for the contact form.
